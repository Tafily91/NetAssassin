NetAssassin 🎯



**Installation and Usage Guide**

git clone https://github.com/Tafily91/netassassin.git

cd netassassin

chmod +x setup.sh netassassin.py

sudo ./setup.sh



💻 Usage

netassassin

netassassin -t 192.168.1.1 -p 1-1024

netassassin -t 192.168.1.0/24 -p 80,443 -a

netassassin --history
